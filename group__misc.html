void Int2Char ( unsigned int  n,
unsigned char  b[2] 
)

Converts unsigned int to unsigned char

Break up an unsigned int (16 bits) into 2 unsigned char (8 bits each). First unsigned char will hold the 8 upper bits while second unsigned char will hold the 8 lower bits. unsigned int is broken to unsigned char so it can be writte through SPI and transmitted through RF.

Parameters
n- source, unsigned int to be broken up
b[]- destination, array of 2 unsigned char to be populated

void ConsoleComment ( void  )

Allows comment in console/PuTTY.

GetChar() will echo back the character received from UART. This allows PuTTY to print whatever you typed. The MCU will not save anything you typed during the call of this function. I wrote this solely to be able to write & document my tests in PuTTY, which I then log everything into a file.
