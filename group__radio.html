void InitRF ( void  )

Initialize RF transceiver.

This function needs to be called before using any of the RF functions.

Note
If you want to use the EEPROM after this function is called, you need to call InitEEPROM() again because EEPROM and RF transceiver both share the SPI. Calling the InitEEPROM() or InitRF() will change the device that SPI is connecting to.
void TransmitPacket ( unsigned char  b[PACKET_SIZE])

Transmit packet through RF.

Transmit an array of size PACKET_SIZE. Fixed size array and it has to be unsigned char. If you want to send unsigned int, use Int2Char() to break it into unsigned chars first, before sending.

Parameters
b[]- array to be sent as payload
void ReceivePacket ( unsigned char  b[PACKET_SIZE])

Receive packet through RF.

Receive packet that is transmitted from other nRF9E5. If you sent a splitted unsigned int from the transmitter, you have to use Char2Int() to join them back into an unsigned int, else you'll just get 2 unsigned char that probably isn't useful for you.

Parameters
b[]- array to be populated
